# Evaluating and Improving SAM 2 Robustness Under Sensor Degradations

This repository contains the source code and replication materials for the **Visual Media Report Assignment (2026)** at the University of Tokyo.

## Environment Architecture
- **OS:** Windows 11
- **CPU Platform:** Intel Host Processor (Pure CPU fallback enforcement)
- **Core Frameworks:** PyTorch (CPU wheel), Ultralytics SAM 2, OpenCV-Python

## Repository Structure
- `run_all.ipynb`: The unified Jupyter Notebook containing the environment check, baseline inference, 3 image degradation simulations (Low-Light, Motion Blur, Gaussian Noise), and the contrast-adaptive CLAHE improvement framework.
- `generative_ai_usage_log.txt`: Institutional log documenting the secure leverage of generative AI utilities.
